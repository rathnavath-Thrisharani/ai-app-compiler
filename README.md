# AI App Compiler

## Overview

AI App Compiler converts natural language requirements into validated application blueprints through a multi-stage compiler pipeline.

The system transforms user prompts into:

* Intent Definitions
* Application Architecture
* Database Schema
* API Schema
* UI Schema
* Runtime Preview

## Architecture

User Prompt
→ Intent Extraction
→ Architecture Generation
→ Schema Generation
→ Validation Engine
→ Repair Engine
→ Runtime Renderer
→ Metrics Collection

## Features

* Multi-Stage Generation Pipeline
* Schema Enforcement
* Validation & Repair Engine
* Clarification Handling
* Runtime Preview System
* Metrics Dashboard
* Component-Based Rendering
* Support for CRM, Ecommerce, Calculator, Hospital, Health Monitoring and School Management Systems

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Google Gemini
* Vercel

## Evaluation Metrics

The system tracks:

* Success Rate
* Failed Requests
* Repairs Triggered
* Request Latency
